# FlowNote

FlowNote は、すべての変更履歴を保持しながら軽量に動作するモダンな Windows 対応メモアプリです。インクリメンタル検索、ショートカット、自動保存、リマインダー/アラーム機能を備えており、誤って削除したメモも履歴から安全に戻せます。

## 主な特徴

- ✅ **自動バージョン履歴**: 保存のたびに履歴を保持し、削除してもワンクリックで復元可能。
- 🔍 **インクリメンタル検索**: 入力と同時に全文検索し、ヒットした文脈をハイライト表示。
- ⚡ **キーボードショートカット**: `Ctrl+N` で新規メモ、`Ctrl+F` で検索、`Ctrl+S` で即時保存、`Ctrl+H` で履歴表示。
- 💾 **自動保存**: 入力停止後すぐにバックグラウンド保存。クラッシュでも安心。
- 🔔 **リマインダー & アラーム**: Windows 通知または音量漸増のアラームを設定可能。再起動後も保持され、停止するまで鳴動します。
- 🌓 **モダン UI**: グラデーション・ガラスモーフィズムを取り入れたモダンな編集体験。
- 🧠 **メモリ使用量を抑制**: メタデータはインデックスファイルに、本文は個別ファイルに保存し必要なタイミングでのみ読み込みます。

## セットアップ

```bash
npm install
```

## 開発サーバーの起動

```bash
npm start
```

Electron ウィンドウが立ち上がります。リマインダーやアラームをテストする際は、Windows 環境での動作を推奨します。

## データ保存先

ユーザーデータは Electron の `app.getPath('userData')/FlowNote` 以下に、以下の構成で保存されます。

```
FlowNote/
├─ index.json            # メタデータ
├─ notes/*.json          # 各メモ本体
├─ history/*.jsonl       # バージョン履歴 (JSON Lines)
└─ reminders.json        # リマインダー定義
```

## リマインダーについて

- **通知タイプ**: Windows のネイティブ通知を使用します。
- **アラームタイプ**: アラーム用ウィンドウと音声を起動し、システムを自動的にミュート解除、音量を段階的に上昇させます。ユーザーが停止するまで鳴り続けます。
- アラーム停止後は元の音量/ミュート状態に戻します。
